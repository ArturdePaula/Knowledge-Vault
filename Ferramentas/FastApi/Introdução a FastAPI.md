FastAPI é um web framework para criação de API's de base python. Moderno, Simples de configurar e usar e também simples de ser colocado em produção. Com sua abordagem simples, ele diminui consideravelmente a quantidade de erros induzidos por humanos, mantendo a robustez necessária para administrar múltiplos acessos e processos. Ele também já vem com documentação gerada automaticamente, que pode ser acessada pelo endereço da aplicação "/docs" para uma interface swagger, mas também tendo a possibilidade de usar o "/redocs", outra interface para documentação, essas interfaces também podem ser usadas para testes nas rotas.

O FastAPI também conta com servidor interno, de capacidade reduzida para testes de desenvolvimento. 

Mas pode ser visualizado na [documentação oficial](https://fastapi.tiangolo.com/) ou então no [repositório](https://github.com/fastapi/fastapi) no Git Hub
