Distribuições Linux são a base do uso do Linux, para começar a usar, precisamos primeiro entender como funciona tudo isso, mesmo de maneira básica. 

## Kernel

Núcleo do sistema operacional, que se comunica diretamente com o Hardware para controle e gerenciamento, como por exemplo o da memória RAM e do Disco. Tudo isso é feito através dos drivers, que são abstrações para o controle direto do Hardware. O Kernel e os drivers operar no Kernel space, que é um espaço de memória resguardado tempo privilégios mais elevados de operação. 

## Shell 

Interface amigável para o usuário que entra em contato com o Kernel para executar comandos. Existem muitos modelos de Shell, como Bash, Zsh e Fish 

## O que é uma distribuição Linux ? 

Uma distribuição Linux é uma implementação do Kernel Linux, mas adições e modificações a mais referentes aquela distribuição. Por exemplo, o Ubuntu, é uma implementação do Kernel Linux com modificações e aplicativos desenvolvidos pela Canonical, e assim é com as outras distribuições também, no final das contas, cada distribuição é um "sabor" diferente do Linux. 

Temos no mundo Linux duas categorias principais de distribuições: 

#### Enterprise 

São distribuições focadas para o uso em servidores. São bastante estáveis, não havendo mudanças bruscas e tendo atualizações criticas constantes para manter o ambiente o mais seguro possível
#### Desktop

São distribuições focadas no usuário final comum, são atualizadas com muito maior frequência para acompanhar as atualizações dos aplicativos que ela contem. Geralmente vem acompanhadas também de interfaces gráficas para facilitar o uso.
